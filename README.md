This package contains five games:

1. Brain-even
2. Brain-calc
3. Brain-gcd
4. Brain-progression
5. Brain-prime

You need to give the correct answer to the game question. Each game includes 3 rounds

## Installation
```
$ composer global require aspov/brain-games
```
